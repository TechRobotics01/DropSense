# V1 control flow logic
so basically in V1 prototype of this project the flow goes something like this 

1. the MPU605 sensor iniitializes
2. the mpu6050 captures the data(acceleration and gyroscope values)  at a veyr high speed
3. these values are then used to calculatte the g-force (the formula used is sqrt(ax^2+ay^2+az^2)/9.81
4. then the thresholds are defined like when the device is at rest the gforce is 1G when it is in freefall the gforce exerted is 0G and when it hits the ground the gforce can be anywhere from 5-15-20Gs
5. after this the DATA IS LOGGED TO THE SD CARD  
6. it tells the time using the millis fucntion and the gForce values calculated as per the formula and then a;sp 3 boolean variables called fall stat and impt for (falling stationary and impact) respectively 
7. this is enough for the prototypes as our goal of finding out the values is ful filled 

AND THIS IS BASICALLY ALL WHICH IS REQUIRED FOR THE PROOF OF CONCEPT 
AFTER THIS WE MOVE ONTO INTGRATION THIS ENTIRE SYSTEM ONTO A PCB

# PCB design 

the pcb consists of these parts 
1. Esp32 S3 wroom 1 module 
2. mpu6050 (smd)
3. sd card holdee (DM3AT)
4. ME6211 voltage regulator 
5. TP4056 battery protection module 
6. Micro usb (for programming the esp32 and also charging the battery)

The 5v from the micro usb port is fed into the voltage regulator module and then the 3v3 volts is fed into the TP4056 module for charging the batter and also for the battery protection then that power is fed into the esp32 s3 wroom 1 module the mpu605 and also the sd card slot for powering the sd card when i'll be connected to the holder

while uploading the code keep in mind to press the boot button then the enable button and then release the boot button first and then release the enable button 

while charging the battery the green led will be on and when done charging and it is on stand by the red led glows 

Many decoupling capacitors are added near the esp32, mpu605 and the sd card holder, and the BMS to clean the high frequenct niose and also provide stable power...

overall the pcb features a small size factor so small so that it can fit inside your pocket easily
